# GENP

### A Procedural Terrain Generation System in Unity

## Overview

GENP is a Unity-based procedural terrain generation project focused on creating diverse and reproducible game worlds through mathematical noise functions and real-time terrain sampling. The project explores algorithmic world generation techniques, balancing randomness with controllable terrain features suitable for gameplay environments.

This project was completed as part of the Winter Project ’25–26 **GENP (Procedural Terrain Generation Game)** under the mentorship of the Game Development Club, IIT Kanpur.
